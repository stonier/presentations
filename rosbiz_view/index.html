Overview

What is ROS?

  • One of many robot glues that emerged in the 2000's
  • Also Microsoft Robotics Studio, OROCOS, Orca, MOOS, YARP, OPROS.
  • Emerged from Stanford in 2007.
  • Migrated to Willow Garage and finally the Open Source Robotics Foundation.
  • Not owned by a company.
  • Not dependent on a university and it's funding imperatives.
Overview

What is ROS Technically?

  • Software Ecosystem
    • Alot of tools for packaging
    • 4000 publicly hosted software packages
    • 4 million package downloads a year
    • Extensive community support for users
    • 40+% growth every year for five years
  • Plumbing
    • Distributed Development Middleware
    • Internal and External Communication SW
    • Development Tools
  • Capabilities
    • Navigation Framework
    • Drivers...
Overview

How Does it Stack Up? - ROS is ì •!

Compared to others (e.g. OPROS, MSRDS...)

  • Crazy growth rate 40+%, must be doing something right
  • Not perfect software, but does all the other things right
    • Fantastic developer tools
    • Huge and easy access to research software
    • Backed it up with the world's leading proof of concept (pr2)
    • Doesn't force big changes to your existing development patterns
  • Doesn't try to be the tool for every job

Overview

How We Use ROS (Yujin)

  • Rapid prototyping and development
    • Our 1st ROS robot migrated from another robot in 1/2 time.
    • Managers were happy.
    • New recruits have less to learn.
  • Less bug hunting
    • Great debugging tools that we never have time to make.
    • I often work with 2-3 other international engineers on bugfixes to core software.
  • Packaging Software
    • Has an entire framework already, we don't need to build.
  • The robotics community
    • Get priveleged priority on fixes/features we need.
    • Large network of people to lean on for expertise.
    • Easier to recruit high quality interns/developers.

Be wary of everything done in-house:


Differentiators are important, but not everything is a differentiator. Too slow.

Important Messages - III


Watch out for fanboys about ROS, OPROS, open source...
